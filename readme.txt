Send every 404 page not found error directly to start page to overcome problems with search engines

== Description ==

If you don't like the typical 404 page than you can redirect all 404 page not found errors permantly to the start page.

This will help you solve problems with external bad links which are not under your influence.

Check out my other [Wordpress Plugins](http://wordpress.org/extend/plugins/profile/1manfactory)

== Installation ==

1. Upload 404-to-start to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
done

== Remove plugin ==

2. Delete plugin through the 'Plugins' menu in WordPress
